#### Causes  

This error occurs when a Payer user requests an action reserved for Owner users.   

For more information on the actions permitted for Payers and Owners, see the [Categories](/guides/users/categories) article.

#### What to do  

A Payer can become an Owner by providing the necessary information and accepting Mangopay’s terms and conditions. For further guidance, see the [Categories](/guides/users/categories#integration) article.
