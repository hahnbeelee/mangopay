#### Causes  

This error is triggered when attempting to process a payment with an `Active` parameter set to `false`.  

The card can be set to inactive if:  
- The end user did not perform a successful pay-in within 24 hours after the registration.  
- There is an implementation issue and the card was deactivated before the payment could be processed.  

#### What to do  

In such cases, end users need to register their card again on the platform.
