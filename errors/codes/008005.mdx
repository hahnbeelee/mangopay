#### Causes  

This error occurs when the card has an abnormal behavior and has therefore been blocked by Mangopay’s Fraud team.  

#### What to do  

In such cases, end users can use a different payment method.  

If the end user insists on the fact that the transaction is legit, the platform can get in touch with Mangopay’s Support team.
