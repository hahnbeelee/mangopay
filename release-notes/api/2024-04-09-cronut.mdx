### DE virtual IBANs now available in Production  

Platforms can now create DE virtual IBANs to receive bank wire pay-ins to a specific wallet.  

On the <a href="/api-reference/banking-aliases/create-iban-banking-alias">POST Create an IBAN Banking Alias</a> endpoint, the following values have been added to the `Country` parameter:  
- DE  
- DK  
- ES  
- GB  
- PL  

<Note>  
**Note â€“ Sandbox only for DK, ES, GB, PL** 
  
Banking alias creation for DK, ES, GB, and PL is currently only available in Sandbox, to enable testing.  
</Note>  

For GB virtual IBANs, the response includes the `LocalAccountDetails` parameter containing an `AccountNumber` and `SortCode` in the local UK format.
