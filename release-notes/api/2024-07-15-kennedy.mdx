### Turkish National ID card now accepted

For all platforms, national ID cards issued by Turkey are now accepted for `IDENTITY_PROOF` type <a href="/api-reference/kyc-documents/kyc-document-object">KYC Documents</a>.

**Note:** Driving licenses issued by Turkey are **not** accepted (but passports are, as for all countries).
