### New statement descriptor for SEPA direct-debit  

Allow for the use of a statement descriptor for SEPA direct debits.  
  <div class="pill pill-orange rn-pill">Fixed</div> 
  
### Corrected issues  
-Fixed a bug with reports that allowed for a desynchronization to happen for Payouts  
-Fixed a bug where in some very rare cases mandates would not be created successfully  
-Fixed a bug in some minor cases whereby you could still refund a pay-in that had been disputed  
-Fixed a bug where some info was not correctly present in the reports for bank wire pay-ins 
  
### Small enhancements  
-Updated the validation service for GB bank accounts to improve reliability  
-Add a specific error when trying to refund a non-successful transfer
