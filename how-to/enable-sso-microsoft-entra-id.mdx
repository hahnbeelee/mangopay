Single sign-on (SSO) lets your team members authenticate their access to the Mangopay Hub via a third-party identity provider.

Setting up SSO can help simplify access management and increase security by enforcing standards.

<Note>  
**Note - Enabling SSO replaces previous credentials** 
  
If you decide to set up SSO, Hub users will no longer be able to authenticate using their previously created passwords.  
</Note>  

<Info>
**Prerequisites**

- An Azure account with an active subscription
- Microsoft Entra ID as your identity provider
- Mangopay registered as an application in your Microsoft Entra ID account (for more information, see the [Microsoft Quickstart guide](https://learn.microsoft.com/en-us/entra/identity-platform/quickstart-register-app))
</Info>

## 1. Configure Microsoft Entra ID for Mangopay

In Microsoft Entra ID, go to _Applications_ > _Identity_ > _Applications_ > _Enterprise applications_ > _All applications_:

- Search for and select the Mangopay app.
- Assign the users or groups that will be able to log in to Mangopay.

In the left navigation pane, select _App registrations_. In _App registrations_, select your Mangopay app, go to the _Overview_ menu:

- Note the Application (client) ID for the next step.


## 2. Retrieve your Client Secret and Client Secret Token

In your Mangopay app registration, go to the _Certificates & secrets_ menu and then the _Client secrets_ tab:

- Create a new Client Secret.
- Note the Client Secret Token for the next step.

Once the Client Secret Token expires, you will no longer be able to connect to your application using the Entra ID SSO. You will have to generate a new token and contact Mangopay to regain access.
</Warning>

## 4. Contact Mangopay

Contact the Mangopay Support team [via the Hub](https://hub.mangopay.com/) (category: My platform account > Hub and dashboard).

Set the subject as:

- Microsoft Entra ID SSO activation

In the description, make sure to include the following information:

- Microsoft Entra ID domain (example.onmicrosoft.com)
- Microsoft Entra ID Client ID
- Microsoft Entra ID Client Secret Token

## 5. Await confirmation to coordinate switchover

When Mangopay activates the Microsoft Entra ID SSO for your application, you will receive a confirmation email.

## 6. Instruct your users to log in with Microsoft Entra ID

On the [Hub login screen](https://hub.mangopay.com/), your users will see a button prompting them to log in with Microsoft Entra ID.
