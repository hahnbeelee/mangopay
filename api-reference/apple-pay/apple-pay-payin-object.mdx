### Description

The Apple Pay PayIn object allows platforms to process payments with Apple Pay.  

<Warning>  
**Caution â€“ Prerequisites to using Apple Pay** 
  
Using Apple Pay requires approval and activation from Mangopay. See <a href="/guides/payment-methods/apple-pay/how-to">How to process an Apple Pay payment</a> for more information.  

The platform also needs to integrate with Apple Pay. For more information, see the <a href="https://developer.apple.com/design/human-interface-guidelines/apple-pay" target="_blank">Apple Pay documentation</a>.   
</Warning>
