### Description
The Google Pay PayIn object allows platforms to process payments with Google Pay.   

<Warning>  
**Caution – Prerequisites to using Google Pay** 
  
Using Google Pay activation from Mangopay. See <a href="/guides/payment-methods/google-pay/how-to">How to process an Google Pay payment</a> for more details and contact our teams <a href="https://hub.mangopay.com/" target="_blank">via the Hub</a>.  

The platform also needs to integrate with Google Pay. For more information, see the <a href="https://developers.google.com/pay/api" target="_blank">Google Pay documentation</a>. 
</Warning>
