### Description
A transfer is a request to relocate funds from one wallet to another.  

<Note>  
**Note â€“ Multi-currency transfers not allowed** 
  
Transfers can only be made between wallets using the same currency.

To convert funds between wallets of different currencies, see <a href="/api-reference/conversions/conversion-object">Conversions</a>. 
</Note>
