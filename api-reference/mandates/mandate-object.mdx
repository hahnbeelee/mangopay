### Description
Mangopay relies on the Mandate object to authorize automatic payments from a Bank Account object to a Wallet.Â   

This payment method is often referred to as SDD (SEPA Direct Debit), but Mangopay also supports the BACS scheme for GBP payments.  

This feature is only available under certain conditions. Contact the Support team <a href="https://hub.mangopay.com/" target="_blank">Mangopay Hub</a> for activation and setup.
