### Description
The Satispay PayIn object allows platforms to process payments made with the Satispay payment method.  

<Note>  
**Note – Activation required** 
  
To activate Satispay for your platform (including Sandbox), contact Mangopay <a href="https://hub.mangopay.com/" target="_blank">via the Hub</a>.  
</Note>
